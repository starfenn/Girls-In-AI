## 一般条件执行  
这个名字是我自己取的，哈哈，因为不知道怎么叫这种简单的用法。   
它格式很简单就是   
如果（满足这个条件), 就（怎么怎么样）if

## 分支执行  
也很简单，用汉语翻译过来就是  
如果（满足这个条件), 就（怎么怎么样）， 否则就（怎么怎么样） if else

## 链式条件执行  
如果存在两个以上的条件，就需要多个if了。   
如果，就；如果，就；如果，就······   
第二个如果的时候就用elif, 即else if的缩写

## 嵌套条件执行  
这个就是一个条件嵌套到另一个条件语句中  
如果，就（如果，就）； 否则就
